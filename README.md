# Pluvia

## Situación

Se requiere un sistema abierto que permita tener registro de la cantidad de agua caída en forma de precipitación y que, adicionalmente, sea accesible a la comunidad a un bajo costo.

## Propuesta

La propuesta esta dividida en varios frentes:

### Creación de pluviómetros

Con herramientas sencillas y material de bajo costo y fácil consecución se puede hacer un pluviómetro con la precisión suficiente para aportar una lectura confiable a la plataforma.

### Ingreso de datos

La información tomada del pluviómetro se ingresaría a una plataforma de toma y almacenamiento de datos a través de WhatsApp.
El detalle de dicho sistema es lo que se detalla en este repositorio Git.
